Script de Limpieza para Windows (PowerShell)

Copia el siguiente código y pégalo en tu consola de PowerShell con permisos de administrador:

# Limpieza segura de archivos temporales y cachés en Windows

# Eliminar archivos temporales del usuario
Remove-Item "$env:TEMP\*" -Recurse -Force -ErrorAction SilentlyContinue

# Eliminar archivos temporales del sistema
Remove-Item "C:\Windows\Temp\*" -Recurse -Force -ErrorAction SilentlyContinue

# Vaciar la papelera de reciclaje
Clear-RecycleBin -Force -ErrorAction SilentlyContinue

# Limpiar caché de actualizaciones de Windows
Stop-Service wuauserv -Force
Remove-Item "C:\Windows\SoftwareDistribution\Download\*" -Recurse -Force -ErrorAction SilentlyContinue
Start-Service wuauserv

# Limpiar carpeta Prefetch
Remove-Item "C:\Windows\Prefetch\*" -Recurse -Force -ErrorAction SilentlyContinue

# Eliminar accesos directos recientes
Remove-Item "$env:APPDATA\Microsoft\Windows\Recent\*" -Recurse -Force -ErrorAction SilentlyContinue

# Limpiar caché de miniaturas
ie4uinit.exe -ClearIconCache

# Borrar historial de PowerShell
Remove-Item (Get-PSReadlineOption).HistorySavePath -ErrorAction SilentlyContinue

# Limpiar caché de navegadores (Chrome y Edge)
$localAppData = $env:LOCALAPPDATA
Remove-Item "$localAppData\Google\Chrome\User Data\Default\Cache\*" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "$localAppData\Microsoft\Edge\User Data\Default\Cache\*" -Recurse -Force -ErrorAction SilentlyContinue

# Limpiar logs de errores del sistema
Remove-Item "C:\Windows\Logs\*" -Recurse -Force -ErrorAction SilentlyContinue

# Limpiar caché de instalación de controladores
Remove-Item "C:\Windows\System32\DriverStore\Temp\*" -Recurse -Force -ErrorAction SilentlyContinue

# Limpiar archivos de diagnóstico
Remove-Item "$env:ProgramData\Microsoft\Diagnosis\*" -Recurse -Force -ErrorAction SilentlyContinue

# Limpiar caché de reportes de fallos
Remove-Item "$env:LOCALAPPDATA\CrashDumps\*" -Recurse -Force -ErrorAction SilentlyContinue

# Eliminar archivos de instalación antiguos de Windows Update
Dism.exe /Online /Cleanup-Image /StartComponentCleanup /ResetBase

# Eliminar puntos de restauración antiguos (mantiene el más reciente)
vssadmin delete shadows /for=C: /oldest

Write-Host "Limpieza completada correctamente."
